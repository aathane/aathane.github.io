bioMérieux — Stage

Context

As part of the Data Science Microbiology team at bioMérieux, I contributed to the development of tools supporting the analysis of microbiological data derived from mass spectrometry (MALDI‑TOF). The team plays a key role in identifying pathogens to support clinical diagnostics and antimicrobial treatment decisions.

Details

  • Web applications for Mass Spectrometry data investigation: designed and deployed Streamlit web applications to compute from and visualize mass‑spectrometry data. Integrated an ML classification model and interactive Plotly visualizations to streamline data analysis for investigation teams. Final users are Data Scientist, Engineers, and Microbiology Engineers.
  • REST API development: developed and deployed a Docker‑containerized FastAPI service, following CI/CD best practices, to expose core data‑processing functions to internal software teams.
  • Python package for taxonomy database quality: created and published a Python library that assesses the completeness and consistency of taxonomy databases, facilitating automated quality checks.

Success

All projects have been completed, tested and deployed, with positive feedback from internal users. The web applications for mass spectrometry data visualization enable investigation teams to better visualize or cross‑reference data. The packaging project enables the quality of several taxonomic databases to be tested, and the deployed API enables software teams to call up computational functions to transform their data. The variety of assignments has enabled me to develop my skills in data visualization, software development and DevOps, to put my theoretical knowledge of ML into practice and to familiarize myself with the microbiology environment.
